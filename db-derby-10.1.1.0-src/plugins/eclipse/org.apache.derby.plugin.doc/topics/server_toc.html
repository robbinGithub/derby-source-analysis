Starting and Stopping the Derby Network Server

The Derby Network Server for the host running Eclipse can be started and stopped within Eclipse. Multiple network servers can be started using this option by changing the port number the Derby Network Server listens on.

Starting the Derby Network Server

Multiple Derby Network Servers can be started on the host running Eclipse using this menu option. Additionally, the Derby Network Server can be started using the ip address of the host running Eclipse to allow remote client access to the network server.

Stopping the Derby Network Server

Any Derby Network Server started from within Eclipse can be stopped using this menu option.
